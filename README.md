# Daily Sentence 
Anki 2.1.X 插件 - 获取Bing每日一句和BBC的今日短语.

## 功能
- 获取BBC的今日短语栏目
- 获取Bing的每日一句栏目

并且导入Anki

## 安装
- 打开Anki输入插件代码: [1004445081](https://ankiweb.net/shared/info/1004445081)
- 重启Anki

## 使用
1. 选择“工具” - “获取每日一句” ：
    1. BBC 今日短语
    2. Bing 每日一句
2. 选择导入目标牌组名字
